# BlackJack_JS

<img width="334" height="250" alt="image" src="https://github.com/user-attachments/assets/74443cdf-30fd-4ad3-8b41-e841f7b84768" />

Pet-project, созданный для изучения основ программирвоания на Java Script.

Краткое описание: браузерная мини-игра, аналог известной карточной игры BlackJack.

Как играть:

- Нажимаешь кнопку "Старт"
- Решаешь, стоит ли взять еще карту.
- Заканчиваешь свой ход, и сверяешь свои карты, с картами диллера.

P.S. Главная цель была - изучение JS, в свзи с чем визуальное оформление хромает.
